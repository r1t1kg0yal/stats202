## Government Deficits ≡ Private Surpluses

```
    THE FUNDAMENTAL INSIGHT
    ═══════════════════════
    
    (S - I) = (G - T)  (closed economy, no foreign sector)
    
    
    Case 1: Balanced Budget (G = T)
    ────────────────────────────────
    
         G - T = 0
         
         ∴ S - I = 0
         ∴ S = I (private sector)
         
         
    Case 2: Government Deficit (G > T)
    ──────────────────────────────────
    
         G - T = +$100 billion
         
         ∴ S - I = +$100 billion
         ∴ S > I (private sector MUST run surplus)
         
         
    Case 3: Government Surplus (G < T)
    ──────────────────────────────────
    
         G - T = -$100 billion
         
         ∴ S - I = -$100 billion
         ∴ I > S (private sector MUST run deficit)
         
         
    ┌─────────────────────────────────────────────────────┐
    │                                                     │
    │  Government deficit = Private sector surplus       │
    │                       (exactly, always)             │
    │                                                     │
```

## The Crowding Out Fallacy

```
╔═══════════════════════════════════════════════════════════════╗
║  MAINSTREAM: "Government deficits crowd out private invest"  ║
╠═══════════════════════════════════════════════════════════════╣
║                                                               ║
║  Story: G - T = $100 (deficit)                               ║
║         → Govt borrows $100                                  ║
║         → Absorbs $100 of "loanable funds"                   ║
║         → Less available for private investment              ║
║         → Interest rates rise                                 ║
║         → Private I falls                                     ║
║                                                               ║
║  Conclusion: Deficits "crowd out" private investment         ║
║                                                               ║
╚═══════════════════════════════════════════════════════════════╝

╔═══════════════════════════════════════════════════════════════╗
║  POST-KEYNESIAN: Accounting shows this is BACKWARDS          ║
╠═══════════════════════════════════════════════════════════════╣
║                                                               ║
║  G - T = $100 (deficit)                                      ║
║       ↓                                                       ║
║  Govt spends $100 more than it taxes                         ║
║       ↓                                                       ║
║  Private sector RECEIVES $100 net income                     ║
║       ↓                                                       ║
║  (S - I) = $100 (accounting identity)                        ║
║       ↓                                                       ║
║  Private sector ACCUMULATES $100 net financial assets        ║
║       ↓                                                       ║
║  This IS the govt deficit!                                    ║
║                                                               ║
║  No "crowding out" - govt deficit CREATES private wealth     ║
║                                                               ║
╚═══════════════════════════════════════════════════════════════╝

    Mechanism:
    ─────────
    
    Govt spends $100 on roads (G)
         │
         ↓
    Construction firms receive $100
         │
         ├─ Pay wages: $70
         ├─ Buy materials: $20  
         └─ Profit: $10
         
    Workers & suppliers receive $100 total

         ↓
    They don't spend ALL $100 immediately
         │
         ↓

         │
         ↓
    Private saving = Govt deficit
```

## Stock-Flow Consistency

```
    BALANCE SHEET - FLOW OF FUNDS LINKAGE
    ═════════════════════════════════════
    
    Balance Sheet (Stock):          Flow of Funds:
    ──────────────────────          ──────────────
    
    Assets  = Liabilities           ΔAssets = ΔLiabilities
            + Net Worth                     + ΔNet Worth
            
    At t=0:                         Period t to t+1:
    
    A₀ = L₀ + NW₀                  ΔA = ΔL + ΔNW
    
    
    Stock-Flow Consistency:
    ──────────────────────
    
    NW₁ = NW₀ + ΔNW
    
    where ΔNW = Saving (≡ Investment)
    
    
    ┌──────────────────────────────────────────────────┐
    │                                                  │
    │  Flows (S, I) accumulate into Stocks (NW, K)   │
    │                                                  │
    │  Every flow MUST come from somewhere            │
    │  Every flow MUST go somewhere                   │
    │                                                  │
    │  ∴ Flows sum to zero (accounting closure)       │
    │                                                  │
    └──────────────────────────────────────────────────┘
```
