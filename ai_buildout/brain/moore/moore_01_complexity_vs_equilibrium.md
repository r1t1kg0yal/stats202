## The Complexity Insight: Sensitive Dependence on Initial Conditions

```
    LORENZ'S BUTTERFLY EFFECT
    ═════════════════════════
    
    Two systems, differing by 0.001% initially:
    
    t=0:   System A: x=1.000000    System B: x=1.000001
           ├─ Nearly identical ─┤
           
    t=10:  System A: x=5.23        System B: x=5.31
           │← Small difference →│
           
    t=20:  System A: x=-12.4       System B: x=+8.7
           │←←← TOTALLY DIFFERENT ─→→→│
           
    Implication for Economics:
    ──────────────────────────
    • Future is inherently UNPREDICTABLE
    • Long-run forecasts are IMPOSSIBLE
    • "Equilibrium" positions are MEANINGLESS
    • Only short-run ordinal changes can be analyzed
```

## Animal Spirits: The Unmeasurable Driver

```
    THE KEYNESIAN BEAUTY CONTEST
    ════════════════════════════
    
    Level 0: Choose the prettiest face
             (naive)
             
    Level 1: Choose what OTHERS think is prettiest
             (one-step ahead)
             
    Level 2: Choose what OTHERS think OTHERS think is prettiest
             (two-step ahead)
             
    Level 3: Choose what OTHERS think OTHERS think OTHERS think...
             (infinite regress)
             
             ↓
             
    Result: NO RATIONAL CALCULATION POSSIBLE
    ──────
    
    • Future depends on expectations of expectations
    • Cannot be reduced to probability distributions
    • Conventional, fragile, self-fulfilling
    • Changes discontinuously ("regime shifts")
    
    
    MODELING IMPLICATION:
    
        Investment = f(π, u, ...) + ε_animal_spirits

                                    │
                             Irreducible volatility
                             Cannot be "explained" by observables

## The Proof: Slutsky's Experiment (1927)

```
    CAN WE DISTINGUISH "STRUCTURE" FROM "NOISE"?
    ═══════════════════════════════════════════
    
    Experiment: Generate random numbers, take moving average
    
    Random:  +2, -1, +3, -2, +1, -3, +2, ...
             ↓ (10-period moving average)
    MA:      ──╱╲──╱──╲────╱╲──╱╲─────
    
    RESULT: Looks exactly like business cycles!
    
    ┌─────────────────────────────────────────────────────────┐
    │                                                         │
    │  You CANNOT tell if a time series is:                  │
    │                                                         │
    │  (a) Deterministic chaotic system, OR                  │
    │  (b) Random walk with colored noise                    │
    │                                                         │
    │  by looking at the data alone!                         │
    │                                                         │
    └─────────────────────────────────────────────────────────┘
    
    Implication: Econometric "discovery" of cycles may be spurious
                 High correlations may be "nonsense correlations"
                 Past patterns will NOT repeat in future
