## Flaw #3: IS-LM is Fundamentally Broken

```
    WHY IS-LM FAILS
    ═══════════════
    
    IS CURVE (Investment = Saving)
    ──────────────────────────────
    
    Defined: Combinations of (i, Y) where S = I
    
    Problem: S ≡ I ALWAYS (accounting identity)
             Not just at special (i, Y) combinations
             
             ∴ Every point is "on" the IS curve
             ∴ IS curve doesn't exist as locus
             ∴ IS curve is meaningless
             
             
    LM CURVE (Money Supply = Money Demand)
    ──────────────────────────────────────
    
    Defined: Combinations of (i, Y) where M^S = M^D
             Assumes M^S set by CB (exogenous)
             
    Problem: M is endogenous (credit-driven)
             CB sets i (BR), not M
             M adjusts to credit demand
             
             ∴ LM curve assumes wrong causal direction
             ∴ LM curve is misleading
             
             
    ┌──────────────────────────────────────────────────┐
    │                                                  │
    │  IS-LM intersection determines (i*, Y*)?        │
    │                                                  │
    │  NO: Both curves are mis-specified              │
    │                                                  │
    │  Reality:                                        │
    │    i set exogenously by CB                      │
    │    Y determined by AD at that i                 │
    │    M adjusts endogenously                       │
    │                                                  │
    │  No "general equilibrium" intersection          │
    │                                                  │
    └──────────────────────────────────────────────────┘
```

## The Aggregation Problem

```
    KIRMAN'S CRITIQUE
    ═════════════════
    
    "The behavior of a set of agents is NOT the behavior
     of a single agent writ large"
     
     
    Example: Consumption Function
    ─────────────────────────────
    
    Individual: C_i = α_i + β_i·Y_i
    
    Aggregate: C = Σ C_i = Σ α_i + Σ β_i·Y_i
    
    Does NOT simplify to: C = α + β·Y
    
    Unless: All α_i identical, all β_i identical
            (Absurd assumption!)
            
            
    ┌──────────────────────────────────────────────────┐
    │                                                  │
    │  Cannot build macro by "scaling up" micro       │
    │                                                  │
    │  Heterogeneity matters                          │
    │  Distribution matters                            │

    │                                                  │

## Data Mining and Statistical Significance Abuse

```
    THE ECONOMETRIC PROBLEM
    ═══════════════════════
    
    Researcher has:
    • 1 data set (say, 100 observations)
    • 50 potential explanatory variables
    • Wants to find "the" model
    
    
    Procedure:
    ──────────
    
    [1] Try Model 1: Y = α + β_1·X_1 + ε
        R² = 0.45, t-stat = 1.2 (not significant)
        → Reject
        
    [2] Try Model 2: Y = α + β_2·X_2 + ε
        R² = 0.52, t-stat = 1.8 (not significant)
        → Reject
        
    ...
    
    [50] Try Model 50: Y = α + β_{50}·X_{50} + ε
         R² = 0.73, t-stat = 3.1 (significant!)
         → PUBLISH!
         
         
    ┌──────────────────────────────────────────────────┐
    │                                                  │
    │  Problem: DATA MINING                           │
    │                                                  │
    │  Tried 50 models, reported best one             │
    │  t-stat meaningless (tried many times)          │
    │  Result is SPURIOUS                              │
    │                                                  │
    │  "If you torture the data enough,               │
    │   it will confess to anything"                  │

    McCloskey & Ziliak Study:
    ─────────────────────────
    
    Examined ALL empirical papers in AER (1980s)
    
    Finding: 96% MISUSE "statistical significance"
    
    Confuse:
    • Statistical significance (is t > 2?)

             Statistically significant
