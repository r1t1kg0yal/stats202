# 00 · The Master Map

> One axiom. Two kinds of inflation. One elastic-money era. A lattice of inverse
> pairs. A handful of policy sign-flips. Read this, and every chapter is a corollary.

---

## 1. The axiom

```
  ┌──────────────────────────────────────────────────────────────┐
  │   MONEY ≡ COLLATERAL ≡ ASSETS ≡ BANK BALANCE-SHEET SPACE       │
  │                                                                │
  │   asset = a promise to be paid in the future                  │
  │   creating it = monetizing that future time for use today     │
  │   new money  = a new asset–liability pair, born from nothing  │
  │               on a bank's balance sheet                        │
  │                                                                │
  │   ∴ INFLATION = sustained expansion in the supply of collateral │
  └──────────────────────────────────────────────────────────────┘
                 created ONLY by balance-sheet expanders:
        commercial banks · shadow banks · offshore (Eurodollar) banks
                 (NOT the Fed — see ch.03)
```

## 2. The master dichotomy — every price move sorts into one of two bins

```
                          A  PRICE  IS  RISING
                                   │
              ┌────────────────────┴─────────────────────┐
              ▼                                           ▼
   ┌────────────────────────┐               ┌────────────────────────────┐
   │   MONETARY INFLATION    │               │   NON-MONETARY INFLATION    │
   │   collateral expanding  │               │   anything else (frictions) │
   ├────────────────────────┤               ├────────────────────────────┤
   │ GREASE                  │               │ FRICTION                    │
   │ self-REINFORCING        │               │ self-CORRECTING ("transitory")│
   │ "higher P CAUSE high P" │               │ "higher P CURE high P"      │
   │ asset yields RISE       │               │ asset yields DON'T rise     │
   │ rising-growth           │               │ falling-growth (Q↓ ⇒ M↓)    │
   │ NEEDS a policy fight     │               │ already tightening itself   │
   └───────────┬────────────┘               └─────────────┬──────────────┘
               ▼                                           ▼
   defeated only by a CONTRACTION              collateral is ALREADY falling
   IN COLLATERAL                               relative to prices → self-quells
```

The single sentence that holds the whole framework together:

```
  ┌───────────────────────────────────────────────────────────────────┐
  │ A monetary inflation requires a contraction in collateral to defeat. │
  │ A non-monetary inflation is already tightening itself, because       │
  │ collateral is already falling relative to prices.                    │
  └───────────────────────────────────────────────────────────────────┘
```

P = M/Q, but **Q cannot fall without M falling** — a shock to real output is a
shock to the promises that back the money, so friction inflations destroy money
even as they raise prices. That is why they are transitory.

## 3. The elastic-vs-hard-money spine (the historical engine)

```
   HARD MONEY  (no money printer)            ELASTIC MONEY (collateral printer)
   deflation is the default state            first sustained non-deflation
        │                                          │
   Gold → Claims on Gold → Claims on    ───►   Claims on COLLATERAL
          Base Money / Reserves                (backed only by promises)

  ░░░░░░░░░░░ pre-1950s ░░░░░░░░░  │  ████████ 1950s–2008 ████████  │  ░░░ 2008+ ░░░
   constant inflation/deflation       the elastic era                collateral
   swings; every "money" was scarce    (liability mgmt, repo,         collapses →
                                        Eurodollar, shadow banks)      back to hard
                                                                       → deflation
```

The money printer is the ability to print **assets** — held only by banks.
Deflation privatizes the system's failures (acute pain); a printer socializes
them as inflation (bearable). A functioning system needs the printer.

## 5. The inverse-pair lattice (the same duality in many costumes)

```
   MONEY  ─────────────────────  VELOCITY
   (inflation concept)           (growth / deflation concept)

   STOCK ─────────────────────── FLOW
   (savings · wealth · valuation) (income · spending · cash flows)

   ASSET PRICE ───────────────── ASSET YIELD
   (expensive = money scarce)    (cheap = money plentiful)

   LONG DURATION ─────────────── SHORT DURATION
   FINANCIAL ECONOMY ─────────── REAL ECONOMY
   INSURANCE (bonds) ─────────── THE THING INSURED (risky activity)
   GREASE (monetary) ─────────── FRICTION (non-monetary)
   ELASTIC MONEY ─────────────── HARD MONEY
   SELF-REINFORCING ──────────── SELF-CORRECTING
   CROWD-IN ──────────────────── CROWD-OUT

   RULE:  you cannot have both sides at once.
          tilt to STOCK  = deflation     tilt to FLOW = inflation
          only NEW MONEY adds to BOTH pans at once.
```

## 6. The policy-inversion payload (the heterodox conclusions)

```
   LESS risk-taking / LESS money            MORE risk-taking / MORE money
   (deflationary)                            (inflationary)
   ◄════════════════════════════════════════════════════════════════►
      Deficit REDUCTION                         Deficit EXPANSION
      QE  (buys safe, adds reserves)            QT  (returns safe assets)
      Rate CUTS (pull flow → stock)             Rate HIKES (pull stock → flow)
      HIGH asset prices (low yields)            CHEAP assets (high yields)

   ⇒ QE is a tighten. Rate hikes are stimulative. Cheap bonds are bullish.
     Bear-steepening is expansionary. The wealth effect is deflationary.
     The Fed's mechanical tools are backwards — it works only via psychology.
```

## 7. The historical timeline

```
  pre-1950s ─┬─ HARD MONEY: gold → claims on gold → claims on base money
             │  constant inflation/deflation swings; no sustained non-deflation
             │
  1950s–60s ─┼─ ELASTIC MONEY emerges: liability management, wholesale finance,
             │  repo, Eurodollars. Banks fund both sides of the balance sheet.
             │
  1960s–70s ─┼─ THE GREAT INFLATION = monetary (offshore/unobservable credit
             │  explosion), NOT velocity. Bond yields finally price inflation.
             │
  1981 ──────┼─ VOLCKER repurposes rates into a signal of policy stance.
             │  The Fed-as-psychology paradigm is born.
             │
  1981–2008 ─┼─ The global dollar system smooths through the Fed's distortions;
             │  the Fed's "mask" handles psychology, the plumbing handles mechanics.
             │
  2008 ──────┼─ GFC = COLLATERAL COLLAPSE. MBS revealed unsafe; only Treasuries
             │  left. The unobservable leg implodes → return to hard money → deflation.
             │
  2010s ─────┼─ POST-CYCLICAL PARADIGM / "Silent Depression". QE deflationary,
             │  secular stagnation. The Fed's mechanics fail; "the Fed does not exist."
             │
  2020+ ─────┴─ COVID fiscal = one-shot asset creation → transitory inflation;
                then FISCAL DOMINANCE. Rate hikes potentially stimulative.
```
