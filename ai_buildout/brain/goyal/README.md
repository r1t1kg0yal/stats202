# The Brain — A Collateral Theory of Everything

This is the structured, ASCII-art reconstruction of Ritik Goyal's macroeconomic
framework: an articulation of how inflation, money, credit, asset prices, and
monetary/fiscal policy actually work, derived from a single axiom and propagated
through every domain of macro.

The framework has an internal name in the source chapters: **CB201**, and a
governing metaphor — **heliocentric central banking** (the Copernican move that
removes the Fed from the center of the monetary universe and puts collateral,
bank balance sheets, and the global dollar system at the sun).

---

## The one axiom everything descends from

```
  ┌──────────────────────────────────────────────────────────────┐
  │   MONEY  ≡  COLLATERAL  ≡  ASSETS  ≡  BANK BALANCE-SHEET SPACE  │
  │                                                                │
  │   An asset is a promise to be paid in the future.              │
  │   Creating it monetizes that time for use TODAY.               │
  │   The only "new money" is a new asset–liability pair created   │
  │   from nothing by a balance-sheet expansion.                   │
  │                                                                │
  │   ∴  INFLATION = a sustained expansion in the supply of        │
  │      collateral. Nothing else. Every other price move is a     │
  │      price move, not inflation.                                │
  └──────────────────────────────────────────────────────────────┘
```

Every chapter below is this axiom applied to one more battlefield.
