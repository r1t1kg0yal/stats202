# 05 · Expensive Assets aren't Inflation

> CORE CLAIM: Expensive assets (low yields) always signal a money SHORTAGE;
> cheap assets (high yields) always signal plentiful money. There is a hard
> tradeoff between STOCK (valuation) and FLOW (income). New money is the only
> thing that adds to both. The wealth effect is therefore deflationary.

Source: [5-expensive-collateral-inflation.md](../../converted/5-expensive-collateral-inflation.md)

---

## The yield identity and what "expensive" means

```
                          Cash Flows
        Asset Yield  =  ──────────────
                         Asset Price

   "Expensive" / "cheap" refer to what you PAY for a stream of cash flows,
   NOT the cash flows themselves. The cash flows are the real wealth.
   More assets ⇒ more competition ⇒ cheaper assets ⇒ MORE income per asset.
   An ideal system: plentiful assets, all CHEAP, all throwing off lots of income.
```

## The stock-flow seesaw (the unifying picture)

```
        ┌──────────────────── STOCK ⇆ FLOW ────────────────────┐
        │                                                        │
        │   STOCK side                          FLOW side        │
        │   ───────────                         ──────────       │
        │   valuation                           income / cash flows│
        │   long duration                       short duration    │
        │   low yield                           high yield         │
        │   low velocity                        high velocity      │
        │   FINANCIAL economy                   REAL economy       │
        │                                                        │
        │        ╲                                  ╱            │
        │         ╲___________  △  ________________╱             │
        │              tilt STOCK = DEFLATION                    │
        │              tilt FLOW  = INFLATION                    │
        │   only NEW MONEY adds weight to BOTH pans at once       │
        └────────────────────────────────────────────────────────┘
```

## The four ways to move the balance

```
   CASE                                    asset price | cash flows | assets vs CF
   ─────────────────────────────────────── ───────────  ──────────   ────────────
   1  new dollar enters ASSET market         RISE         unchanged    more expensive
   2  new dollar buys a WIDGET               unchanged      RISE        cheaper
   3  cut spending to BUY assets              RISE          FALL        more expensive
   4  sell an asset to SPEND                  FALL          RISE        cheaper
   ───────────────────────────────────────────────────────────────────────────────
   Rate cuts / QE behave like case 1/3 (weight → STOCK = deflationary).
   Issuance / dissaving behave like case 2/4 (weight → FLOW = inflationary).
```

## Rate cuts just frontload the same wealth (a mechanical example)

```
   2y note, $100 face                yield 11.8%  →  price $80   (Fed holds ~11%)
   Fed surprise-cuts to ~5%          yield  5.4%  →  price $90

   the $100 redemption DIDN'T change. The cut just pulled value accrual to the
   PRESENT, worsening future returns. Weight moved FLOW → STOCK. No new wealth;
   only a worse complexion of return.
```

## Cheap assets = liquid REAL conditions (the liquidity inversion)

```
   high asset prices  ──►  "liquidity" in the financial sense (lots of money-like
                            paper)  BUT  poor liquidity CONDITIONS in the real
                            economy (people pay a PREMIUM for liquidity ⇒ liquidity
                            is scarce ⇒ spending/income poor)

   measure liquidity as the NUMBER OF VEHICLES to save in, NOT the number of
   dollars chasing them. Cheap assets ⇒ assets compete with a wide menu (loans,
   spending) ⇒ HIGH nominal growth.
```

> Prometheus: "The second you invest in a financial asset instead of spending it
> into the economy, you're SHORT spot inflation" — and you help cause that outcome.

## Equities prove it: the 1970s

```
   Decade   S&P500   real GDP   inflation
   1970s     5.9%      3.2%       6.7%     ← BEST real growth, WORST equity return
   1980s    17.3%      3.1%       4.8%
   1990s    18.0%      3.2%       2.3%
   2000s    -1.0%      1.9%       2.3%
   2010s    13.4%      2.3%       1.7%     ← weak growth, fat equity return

   high nominal growth ⇒ high collateral creation ⇒ every asset constantly
   re-rated CHEAPER as more assets are produced to compete with it.
```

## Crowd-IN, not crowd-out (the Treasury-issuance case)

```
   CONVENTIONAL (crowd-OUT, a myth here):
     gov't borrows → long rates ↑ → private borrowing cost ↑ → less activity

   THIS FRAMEWORK (crowd-IN):
     gov't borrows → private SURPLUS + new collateral → system can borrow MORE
     at the higher yields. One person's borrowing = another's collateral.

   ⇒ a rise in long yields CAN'T be auto-read as tightening — conditions must
     have LOOSENED for yields to rise. Treasury issuance is always a LOOSENING
     for the real economy (bearish assets, bullish the economy — two sides of one coin).
```

## Bonds as insurance (bridges to ch.06/07)

```
   when INSURANCE (safe bonds) gets EXPENSIVE (low yields), you either:
     (a) allocate MORE to insurance (less for risk), or
     (b) run risky activity UNDER-insured
   both are bad ⇒ worse portfolio returns ⇒ LOW yields crowd OUT risk.
   ⇒ to crowd IN risk, make insurance CHEAP — i.e. RAISE yields.
```

## The wealth effect is deflationary

```
   "feeling wealthier" (QE inflates asset PRICES, not cash flows) ≠ "being wealthier"
   expensive assets ⇒ low income-producing potential ⇒ a DEFLATIONARY signal.
   Empirically, expensive assets never coincide with high inflation. The Fed's own
   research: the wealth channel contributes little; asset declines and inflation
   are two sides of the SAME stock→flow coin.
```

## The path-to-recession, re-ordered

```
   CONSENSUS (Andy/Bob): rates↑ → stocks↓ → demand↓ → earnings↓ → jobs↓ → inflation↓
   THIS FRAMEWORK:       rates↑ → BEAR STEEPEN (asset supply ↑, re-rate lower)
                                  = EXPANSIONARY, not recessionary
     a bear steepening is offset by the feasibility of creating risky assets.
     a recession needs an asset CONTRACTION — which is what curve INVERSION is
     (shortage of long-duration assets vs short-rate cost). Negative wealth
     effect = "Risk Charity" for everyone short assets.
```
