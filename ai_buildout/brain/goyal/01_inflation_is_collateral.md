# 01 · Inflation is Collateral

> CORE CLAIM: Sustained inflation is a generalized rise in prices that can only
> be self-reinforcing under one condition — an increasing supply of collateral.
> Money is the liabilities that match the supply of assets. More money requires
> more assets. Only commercial banks create assets.

Source: [1-inflation-is-collateral.md](../../converted/1-inflation-is-collateral.md)

## The axiom, as a blueprint

```
  ┌─────────────────────────── WHAT MONEY IS ───────────────────────────┐
  │                                                                      │
  │   ASSET  = a promise to be paid in the FUTURE                        │
  │            │                                                         │
  │            ▼  monetizes that time...                                 │
  │   ...for use TODAY  → the matching LIABILITY is spendable money       │
  │                                                                      │
  │   MONEY = the aggregate set of liabilities that match the aggregate   │
  │           supply of assets (collateral)                              │
  │                                                                      │
  │   ⇒ inflation needs more money ⇒ more money needs more assets         │
  │   ⇒ INFLATION = exclusively the creation of new collateral            │
  └──────────────────────────────────────────────────────────────────────┘
```

## How a bank makes money out of nothing (the only money printer)

```
        COMMERCIAL BANK                         A new asset–liability pair
   ┌───────────────┬───────────────┐            is created simultaneously,
   │    ASSETS     │  LIABILITIES   │            from nothing. No cash moves.
   ├───────────────┼───────────────┤
   │ + New Loan    │ + New Deposit  │   ◄── the deposit IS the borrower's money
   │   (a promise) │   (spendable)  │       the loan IS the promise backing it
   └───────────────┴───────────────┘
        this is the actualization of ELASTIC money:
        money backed only by money-in-the-future.
```

## The two kinds of inflation (the master fork)

```
   MONETARY INFLATION                  │   NON-MONETARY INFLATION
   ─────────────────────────────────── │ ───────────────────────────────────
   produced by production of ASSETS    │   produced by ANYTHING else
   inflation of GREASE                 │   inflation of FRICTION
   self-REINFORCING                    │   self-CORRECTING ("transitory")
   "higher prices CAUSE higher prices" │   "higher prices CURE higher prices"
   asset yields RISE                   │   asset yields DON'T rise
   requires a policy response          │   requires no policy response
```

## Why collateral is elastic: risk crowds in risk

```
   UNSECURED loan                          COLLATERALIZED loan (repo)
   ───────────────────────────            ───────────────────────────────
   $10 expected-loss limit                 same $10 limit, but the loan is
   $100 loan to a 10%-default               backed by $100 of Treasuries
   borrower → limit MAXED OUT               (≈0% default) → limit UNTOUCHED
   → balance sheet CANNOT expand            → balance sheet can KEEP expanding

   Collateral circumvents trust through verification. Repo unlocked a vast
   pool of cash to be lent safely. Safe assets make the BEST collateral.
```

```
        ┌──────────────── THE GREASE ENGINE (reinforcing loop) ───────────────┐
        │                                                                      │
   collateral ──► bank b/s capacity ──► money creation ──► more assets ──┐     │
   supply ▲           (new A–L pairs)                                    │     │
          │                                                              │     │
          └──────────────────────────────────────────────────────────────┘     │
        │              "RISK CROWDS IN RISK" — assets beget assets             │
        └──────────────────────────────────────────────────────────────────────┘
```

## Hard money fails

```
   GOLD ──► CLAIMS ON GOLD ──► CLAIMS ON BASE MONEY ──► CLAIMS ON COLLATERAL
   └─────────── hard (scarce) ───────────┘              └── elastic (printable) ──┘

   In a hard-money system there is NO money printer. Without a printer the
   system is always exposed to deflation — and deflation (privatized failure)
   is more painful than inflation (socialized failure). The 1950s+ elastic era
   was the first sustained stretch without constant deflationary spirals.
```

## Grease, not friction

```
   FRICTIONS (war, commodity shock, supply chain, demographics) raise prices.
   P = M / Q.  If Q falls, P rises... BUT:

      Q cannot fall without M also falling
      (a real-output shock breaks the promises that back the money)

   ⇒ a friction "inflation" is the market WORKING OUT that demand must be
     destroyed. Every non-monetary inflation was ultimately transitory; every
     transitory inflation was non-monetary. Asset yields don't rise because the
     supply of assets is still scarce.

   FRICTION-led = falling-growth inflation   |   GREASE-led = rising-growth inflation
```

> Count Draghula's razor: "Either you are bullish growth and inflation is a
> problem, OR you are negative growth and inflation isn't a problem. You cannot
> believe both." (Friction inflations are growth-negative ⇒ self-quelling.)

## Bond yields are a monetary-inflation detector

```
   supply of assets ABUNDANT  ──►  money abundant  ──►  monetary inflation
                                                        + bonds must compete
                                                        with more assets
                                                        ──►  YIELDS RISE
   supply of assets SCARCE    ──►  non-monetary price spike ──► YIELDS FLAT
                                                        (bonds are collateral;
                                                         scarcity keeps them bid)
```

Bond yields lifted off only once in two centuries — the 1960s–70s monetary
Great Inflation. Every other "inflation" was transitory, and yields ignored it.
