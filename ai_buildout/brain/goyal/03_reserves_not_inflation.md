# 03 · Reserves aren't Inflation

> CORE CLAIM: Bank reserves are not collateral, so they are not money, so they
> are not inflation. The Fed creating reserves is an asset SWAP for the private
> sector, not an expansion. The Fed has no money printer. The GFC was a dollar
> run (collateral run), not a deposit run — and reserves can't fix that.

Source: [3-reserves-aren't-inflation.md](../../converted/3-reserves-aren't-inflation.md)

## The two-tier monetary system

```
        FED                BANK                       NON-BANK
   ┌─────┬───────┐    ┌─────────┬─────────┐      ┌──────────┬──────┐
   │Asset│Reserves│   │Reserves │Deposits │      │ Deposits │      │
   └─────┴───────┘    │Loans/Sec│Borrow/Eq│      └──────────┴──────┘
                      └─────────┴─────────┘
        ▲                  ▲          ▲                 ▲
        └── reserves live  ┘          └── deposits live ─┘
            ONLY here                     here

   Reserves can sit in exactly 4 places: at banks (IORB), at the RRP, in the
   Treasury's account (TGA), or as physical cash. NONE of these is a private-
   sector asset–liability EXPANSION. There is no path for a reserve to jump to
   the asset side of a NON-bank. Reserves cannot enter the real economy.
```

## A loan creates money; a reserve swap does not

```
   1) BANK MAKES A LOAN  (inflationary)        2) FED CREATES RESERVES (a swap)
   ┌──────────┬───────────┐                    Bank loses a Treasury,
   │ +Loan    │ +Deposit  │  new A–L pair       gains a Reserve in its place.
   └──────────┴───────────┘  collateral ↑       Private-sector collateral does
   reserves played NO part                       NOT rise. Often it FALLS, because
   (banks can't "lend out reserves")             a Treasury is BETTER collateral
                                                 than a reserve.
```

> Count Draghula: "Bank lending is limited by regulatory CAPITAL, not reserves."
> Capital = equity, not reserves. Reserves aren't capital because reserves aren't
> collateral. Banks find the reserves *after the fact* to meet regulations.

## Collateral has three axes — reserves score badly on one

```
                    QUALITY                          UTILITY
        ┌───────────────────────────┐    ┌──────────────────────────────┐
        │  LIQUIDITY (convert→cash)  │    │  QUALITY  +  ACCESSIBILITY    │
        │       ×                     │    │              (who may hold it)│
        │  CREDITWORTHINESS (haircut)│    │                               │
        └───────────────────────────┘    └──────────────────────────────┘

   Reserves & Treasuries tie on QUALITY (both riskless, liquid).
   But on ACCESSIBILITY:
        TREASURIES → holdable by ANYONE, anywhere; collateral everywhere
        RESERVES   → holdable ONLY by banks that bank with the Fed
   ⇒ reserves are far LESS USEFUL collateral. The market proves it: 4-week
     T-bills trade at a PREMIUM to the reserve floor (RRP). Bills > reserves.
```

## The farm fallacy (Lyn Alden) — reserves aren't the base; collateral is

```
   "Food comes from the FARM (reserves), not the grocery store (deposits)."
   ──────────────────────────────────────────────────────────────────────
   Wrong base. The real monetary base is COLLATERAL, not reserves:

        FED's farm (reserves)  ─┐
                                ├─► but most money is created on the
        PRIVATE farm (collateral)┘    PRIVATE collateral farm, with no care
                                       for reserves at all (offshore, repo,
                                       shadow banks). Reserves don't back the
                                       full supply of money. Collateral does.
```

## Pre/Post-2008: reserves never bound the system

```
   PRE-2008 (limited reserves)            POST-2008 (ample reserves)
   ┌───────────────────────────┐         ┌───────────────────────────────┐
   │  ╭──────────────────────╮ │         │  ╭───────────────────────────╮ │
   │  │  reserve-LESS actual  │ │         │  │ reserves-based THEORETICAL │ │
   │  │  total activity       │ │         │  │ allowed activity (huge)    │ │
   │  │   ╭───────────────╮   │ │         │  │     ╭───────────────╮      │ │
   │  │   │reserve-allowed│   │ │         │  │     │ ACTUAL activity│      │ │
   │  │   ╰───────────────╯   │ │         │  │     ╰───────────────╯      │ │
   │  ╰──────────────────────╯ │         │  ╰───────────────────────────╯ │
   └───────────────────────────┘         └───────────────────────────────┘
   reserve limit never bound             reserve limit STILL doesn't bind —
   (offshore creation dwarfed it)        banks just don't WANT to create money
```

The GFC wasn't that the Fed's farm was too small. It was that a huge chunk of
"safe" MBS collateral was revealed worthless, so the **private** farm was too
small. A **dollar run / collateral run**, not a deposit run. More reserves can't
fix a collateral shortage — and QE, by taking Treasuries, can make it worse.

## Friedman's interest-rate fallacy (foreshadows ch.05/07)

```
   conditions LIQUID  ──►  yields HIGH (asset prices low)
   conditions ILLIQUID ──►  yields LOW  (asset prices high)

   Causation runs CONDITIONS → RATES, not rates → conditions.
   ⇒ reserves are "liquidity" (liquid instruments) but their presence makes
     "liquidity CONDITIONS" WORSE: fewer long bonds, more reserves, inflated
     assets, lower yields. High asset prices = poor liquidity conditions.
```
