## Core Claim

Crises are systemic. They are not random shocks hitting an otherwise stable economy. The common root is the development of increasingly optimistic expectations alongside real and financial structures that cannot validate those expectations. In open economies, Harvey locates the break in one or more of three tension points.

```
┌──────────────────────────────────────────────────────────────────────┐
│                  HARVEY CRISIS ARCHITECTURE                          │
├──────────────────────────────────────────────────────────────────────┤
│  TP1: Currency-forecast divergence                                   │
│       AER moves away from the mental-model shadow value               │
│                                                                      │
│  TP2: Financial-returns divergence                                   │
│       Financial asset returns outrun real investment returns          │
│                                                                      │
│  TP3: Financial fragility                                            │
│       Debt/income, maturity mismatch, and FX mismatch rise            │
│                                                                      │
│  Any one can snap first. Once one snaps, contagion can trigger the    │
│  others.                                                             │
└──────────────────────────────────────────────────────────────────────┘
```

## Generic Crisis Sequence

From the broader crisis paper:

```
Optimistic forecasts accumulate
  + economy/finance cannot validate them
        ↓
SHOCK
  agents realize expectations are out of line with reality
        ↓
NEGATIVE REPERCUSSIONS
  selling, default, devaluation, credit withdrawal
        ↓
CONTAGION
  losses and panic spread to linked balance sheets and markets
```

The shock is not arbitrary. It is the moment when the accumulated gap becomes visible enough that agents abruptly reprice the world.

## TP1: Currency-Forecast Divergence

```
┌──────────────────────────────────────────────────────────────┐
│       R1: Asset/Currency Bandwagon                           │
│                                                              │
│   Domestic asset values rise                                 │
│       │                                                      │
│       ▼                                                      │
│   Total net demand for domestic assets rises                 │
│       │                                                      │
│       ├──→ Portfolio inflows rise ────────────────┐          │
│       │                                           │          │
│       ▼                                           │          │
│   Bandwagon purchases rise                        │          │
│       │                                           │          │
│       ▼                                           │          │
│   Domestic currency appreciates                   │          │
│       │                                           │          │
│       ▼                                           │          │
│   Foreign-currency value of domestic assets rises ┘          │
└──────────────────────────────────────────────────────────────┘

TENSION:
────────
Actual exchange rate - mental-model shadow forecast
```

Under a float, the spot exchange rate can rise above what the mental model would justify. Under a peg, the bandwagon can make the peg look sustainable even while the shadow forecast is deteriorating. The larger the gap, the smaller the catalyst needed to trigger collapse.

## TP2: Financial-Returns Divergence

```
┌──────────────────────────────────────────────────────────────┐
│       B1: Real Investment Saturation                         │
│                                                              │
│   Expected profit from investment rises                      │
│       │                                                      │
│       ▼                                                      │
│   Physical investment rises                                  │
│       │                                                      │
│       ▼                                                      │
│   Stock of physical capital rises                            │
│       │                                                      │
│       ▼                                                      │
│   Marginal expected profit from new capital falls ─────┐     │
│                                                        │     │
│       ┌────────────────────────────────────────────────┘     │
│       ▼                                                      │
│   Real return anchor weakens                                 │
└──────────────────────────────────────────────────────────────┘

TENSION:
────────
Financial asset return expectations - real investment return capacity
```

Financial asset values can keep rising through the bandwagon while the real economy's capacity to generate returns is already decelerating. The gap between financial returns and real returns becomes a crisis point.

## TP3: Financial Fragility

Minsky enters Harvey through debt validation. Good times convince borrowers and lenders that old margins of safety were too conservative.

```
Expansion / asset boom
  → debt repayment looks easy
  → margins of safety fall
  → debt/income rises
  → short-term funding share rises
  → foreign-currency borrowing rises
  → fragility rises
```

| Fragility dimension | Open-economy crisis risk |
|---|---|
| Debt/income | Cash flows cannot validate liabilities |
| Maturity mismatch | Short funding must be rolled in stressed markets |
| Locational mismatch | Domestic-currency depreciation raises foreign-currency debt burden |

```
BREAKS WHEN:
────────────
Currency falls or asset values fall
  → repayment capacity is questioned
  → lenders withdraw
  → borrowers liquidate
  → currency/asset fall accelerates
```

## Complete Open-Economy Crisis Map

```
Financial liberalization / optimistic catalyst
        ↓
Portfolio inflows
        ↓
Asset values rise ←→ currency rises
        ↓              ↓
Bandwagon loops strengthen
        ↓
┌───────────────────────┬────────────────────────┬───────────────────────┐
│ TP1                   │ TP2                    │ TP3                   │
│ AER exceeds shadow    │ financial returns      │ debt, maturity, and   │
│ forecast              │ exceed real returns    │ FX mismatch rise      │
└───────────┬───────────┴───────────┬────────────┴───────────┬───────────┘
            │                       │                        │
            └──────────── any tension snaps first ───────────┘
                                    ↓
                         shock → repercussions → contagion
```

## Episode Classification

| Episode type | Harvey read |
|---|---|
| Mexico 1994 | Currency-forecast divergence plus financial fragility |
| Asian 1997 | Financial-returns divergence plus financial fragility |
| Pure panic | One tension snaps but balance-sheet contagion remains contained |
| Full crisis | Multiple tensions snap and credit/currency feedback loops reinforce |

## Operational Checklist

For a country or currency:

1. Score TP1: Is AER diverging from the mental-model shadow forecast?
2. Score TP2: Are financial asset returns outrunning real investment/profit capacity?
3. Score TP3: Are debt/income, maturity mismatch, or FX-denominated liabilities rising?
4. Identify which tension can snap first and what catalyst is sufficient.
5. Ask whether the first break transmits to the other two tension points.
