# Harvey Module 04: BOP and Trade-Finance Reconciliation

> **Source:** `pieces/Harvey_ExchangeRatesPostKeynesian.md`  
> **Use:** Resolves the apparent contradiction between Post-Keynesian trade theory and Post-Keynesian exchange-rate theory.

---

## Core Claim

Post-Keynesian trade theory and Post-Keynesian FX theory can both be true once the capital account is split into two different markets. Trade imbalances are driven by absolute advantage and financed by endogenously created trade credit. Exchange rates are driven by autonomous speculative/portfolio capital flows. The missing distinction is trade-finance capital versus speculative financial capital.

```
┌──────────────────────────────────────────────────────────────────────┐
│                  THE RECONCILIATION                                  │
├──────────────────────────────────────────────────────────────────────┤
│  Trade flows                                                         │
│    → driven by absolute advantage, technology, cost, income           │
│    → financed by trade credit                                        │
│                                                                      │
│  Exchange rates                                                      │
│    → driven by speculative/portfolio financial capital                │
│    → set the price at which trade occurs                             │
│                                                                      │
│  Therefore: persistent trade imbalance can coexist with capital-flow  │
│  dominated FX.                                                       │
└──────────────────────────────────────────────────────────────────────┘
```

## The Problem Harvey Solves

Two Post-Keynesian claims appear to collide:

| Theory | Claim | Apparent problem |
|---|---|---|
| Trade theory | Absolute advantage determines trade balances | If a country should run a trade surplus, the capital account must offset it |
| FX theory | Portfolio capital flows determine exchange rates | Why should capital flows cooperate with the trade prediction? |

Without a trade-finance distinction, the balance of payments seems to force one theory to dominate the other.

## Capital Account Split

Harvey separates capital flows into two different objects:

```
K_in  = tK_in  + sK_in
K_out = tK_out + sK_out

tK = trade-finance capital
sK = speculative / portfolio financial capital
```

| Flow type | Function | Market logic |
|---|---|---|
| `tK` | Credit, insurance, guarantees, working capital used to finance trade | Endogenous to actual trade transactions |
| `sK` | Portfolio allocation seeking expected capital gain / asset return | Drives exchange-rate pricing |

The FX theory governs `sK`, not the full capital account. The trade-finance component exists because exchange takes time and most trade transactions require credit.

## Endogenous Trade Finance

```
Importer wants foreign goods
  → seller/bank/insurer extends credit or guarantee
  → purchasing power is created for the transaction
  → goods move
  → trade imbalance can exist without forcing AER to BTER
```

If importers had to compete for a fixed supply of foreign currency, exchange-rate adjustment would tend to restore balanced trade. But trade finance expands the effective supply of transaction funding alongside trade demand.

```
Foreign producer gains competitiveness
  → domestic imports rise
  → trade-finance credit rises to fund imports
  → current-account deficit appears
  → AER need not move
  → BTER moves, but AER remains set by portfolio capital
```

## Balance-of-Payments Identity

Harvey's full condition:

```
Exports + trade-finance inflows + speculative inflows
  =
Imports + trade-finance outflows + speculative outflows
```

If speculative flows are balanced by portfolio-return arbitrage:

```
sK_in = sK_out
```

then the current-account imbalance is matched by the trade-finance imbalance:

```
Exports - Imports = trade-finance outflows - trade-finance inflows
```

That is the key move. The capital account offsets the current account because credit is created inside trade, not because portfolio investors decided to make trade theory true.

## What Changes In Analysis

Before Harvey's reconciliation:

```
Current-account deficit
  → must imply portfolio capital surplus
  → FX theory and trade theory conflict
```

After Harvey's reconciliation:

```
Current-account deficit
  → matched by trade-finance capital surplus
  → portfolio capital can still set AER independently
```

## Practical Implications

| Question | Harvey answer |
|---|---|
| Can absolute advantage create persistent deficits? | Yes, if trade finance is available. |
| Does a trade deficit require speculative portfolio inflows? | No. It requires financing, but that financing can be transaction-specific trade credit. |
| Does trade affect FX? | Usually indirectly, through the mental model of portfolio investors. |
| What happens when trade finance tightens? | Trade curves shift inward; trade volume can collapse even if competitiveness is unchanged. |
| Why did global trade fall during financial crises? | Not only demand; trade credit availability also contracts. |

## Operational Checklist

For BOP work:

1. Split capital flows into `tK` and `sK` before drawing conclusions.
2. Use absolute advantage, income, and credit availability to explain trade flows.
3. Use expected portfolio return and mental-model forecasts to explain AER.
4. Treat AER vs BTER divergence as normal, not as an arbitrage failure.
5. Watch trade-finance conditions as a volume constraint on global trade, especially during credit stress.
